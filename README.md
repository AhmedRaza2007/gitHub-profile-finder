# GitHub Profile Finder 🔍

A simple and responsive web application that allows users to search for any GitHub profile and view their public information, including profile details and repositories.

## 🚀 Live Demo

https://git-hub-profile-finder-rust.vercel.app

## 📂 GitHub Repository

https://github.com/AhmedRaza2007/gitHub-profile-finder

## ✨ Features

* Search any GitHub user by username
* Display profile picture, name, username, and bio
* Show followers, following, and public repository count
* Display user's public repositories
* Open repositories directly on GitHub
* Loading state while fetching data
* Handle invalid usernames with a user-friendly error message
* Search using both button click and Enter key
* Responsive design for desktop and mobile devices

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6+)
* GitHub REST API

## 📖 What I Learned

Through this project, I learned:

* Working with APIs using `fetch()`
* Using `async/await` for asynchronous JavaScript
* Handling API errors and 404 responses
* Dynamically rendering data to the DOM
* Working with arrays and looping through repository data
* Building responsive user interfaces

## 🔗 GitHub API Endpoints

### User Profile

https://api.github.com/users/USERNAME

### User Repositories

https://api.github.com/users/USERNAME/repos

## 📸 Preview

Search for any GitHub username and instantly view:

* Profile Information
* Followers & Following Count
* Public Repositories
* Repository Links
* Repository Languages
* Star Counts

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/AhmedRaza2007/gitHub-profile-finder.git
```

2. Open the project folder

```bash
cd gitHub-profile-finder
```

3. Run `index.html` in your browser
